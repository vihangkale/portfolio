# Portfolio Site

A modern professional portfolio built with Next.js, TypeScript, and Tailwind CSS.

## Setup

1. Open a terminal in `/Projects/portfolio-site`
2. Run `npm install`
3. Run `npm run dev`
4. Visit `http://localhost:3000`

## Files to update

- Place your profile photo at `public/profile-photo.svg` or replace it with `public/profile-photo.jpg`.
- Place your resume file at `public/resume.pdf` for the View/Download buttons to work.

## Features

- Fast and SEO-friendly Next.js app
- Clean minimal design
- Responsive layout for mobile, tablet, and desktop
- Professional portfolio sections: Hero, About, Skills, Experience, Projects, Resume, Contact
- Reusable component structure
- Accessible semantic HTML
- Tailwind CSS styling
